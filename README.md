-> Brave : Full support      
-> Google Chrome : Partial support (manifest v2 moment)      
-> Microsoft Edge : Partial support     
-> Librewolf : Untested      
-> The Mullvad Browser : Untested

-> User agent randomization     
-> Swapping some features of navigator      
-> Canvas and webgl randomization
